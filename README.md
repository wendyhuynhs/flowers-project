# flowers-project

Available Scripts:

Npm install

In the project directory, run:

1. Npm Install

   - to install dependencies

2. Npm Run Build

   - builds the app for production

3. Npm start

   - starts your server on localhost:3001

4. Npm run dev

   - starts your development server
   - Open http://localhost:3000/ to view the UI

5. Npm test
   - Runs tests
